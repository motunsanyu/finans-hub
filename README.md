# Proje
- Bu proje kişisel bir web app olarak tasarlanmıştır. 
- Projemiz cloudflare üzerinden yayınlanmaktadır. 

# Başlıca Özellikler
- Piyasa    (usd, eur, xau, btc, brent, silver vb...)
- Kasa      (Aylık gelir-gider takibi)
- Yakıt     (Yakıt fiyat bilgisi ve yakıt alım kayıt sistemi)
- Taksit    (Yapılan taksitli harcamaların takibi)
- Zaman     (Geri sayım yapabilen kayıt sistemi)
- Lig       (Türkiye Süper Ligi Puan durumu ve fikstür takip sistemi)

# Açıklama
✅ MEVCUT ÖZELLİKLER
🔐 1. Güvenlik Sistemi

4 haneli PIN ile uygulama kilidi
İlk açılışta PIN tanımlama
Splash screen animasyonu

💰 2. Piyasa Takibi (Ana Sayfa)

Piyasa sekmesi: USD, EUR, Gram Altın, BTC, BIST100, Brent Petrol
Coinler sekmesi: 17 farklı kripto para
Canlı altın borsası detay modalı (22 ayar, çeyrek, yarım, tam altın)
Altın hesap makinesi (alış/satış hesaplama)
Tema değiştirme (Klasik/Modern)

📊 3. Açık Pozisyon Yönetimi

USD, EUR, Altın, BTC, TRY pozisyonları
Borç (-) veya Varlık (+) olarak girme
Anlık TRY ve USD karşılığı hesaplama
Önceki güne göre değişim takibi

💳 4. Kasa Modülü

Aylık gelir-gider kayıt sistemi
12 aylık özet tablosu
Kategori bazlı grafik gösterim

⛽ 5. Yakıt Modülü

Güncel yakıt fiyatları (API)
Yakıt alımı kayıt sistemi
L/100km tüketim hesaplama
100km maliyet analizi

💳 6. Taksit Takibi

Çoklu profil desteği (2 profil)
Taksit detayları (tutar, tarih, durum)
Ödendi/Ödenmedi işaretleme
Kalan borç özeti

⏰ 7. Zaman Modülü

Geri sayım sistemi
Tarih arası gün farkı hesaplama
Geçmiş kayıtlar kartları

⚽ 8. Süper Lig Modülü

Puan durumu (ESPN API)
Haftalık maçlar (geçmiş + gelecek)
Canlı skorlar (60 saniyede otomatik yenileme)
Fikstür sistemi (34 haftalık sezon, hafta hafta gezinme)


🎨 TASARIM ÖZELLİKLERİ
✅ Dual Theme System: Binance tarzı "Klasik" + Minimal "Modern"
✅ Responsive Design: Mobile-first yaklaşım
✅ PWA Ready: Service Worker + Manifest
✅ Smooth Animations: Geçişler ve mikro-animasyonlar
✅ Dark Mode Only: Gece teması odaklı
